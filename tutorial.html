Speech Recognition (Offline)

Paraformer — Chinese ASR

from funasr import AutoModel

model = AutoModel(
    model="paraformer-zh",
    vad_model="fsmn-vad",
    vad_kwargs={"max_single_segment_time": 60000},
    punc_model="ct-punc",
    # spk_model="cam++",  # uncomment for speaker diarization
)
res = model.generate(input="audio.wav", batch_size_s=300, hotword='魔搭')
print(res)
Combined with vad_model, any length audio is supported. Parameters: batch_size_s (dynamic batch in seconds), hotword (recognition keywords).

model = AutoModel(

Speech Recognition (Streaming)

from funasr import AutoModel
import soundfile

chunk_size = [0, 10, 5]  # [0, 10, 5] 600ms, [0, 8, 4] 480ms
encoder_chunk_look_back = 4
decoder_chunk_look_back = 1

model = AutoModel(model="paraformer-zh-streaming")

wav_file = f"{model.model_path}/example/asr_example.wav"
speech, sample_rate = soundfile.read(wav_file)
chunk_stride = chunk_size[1] * 960  # 600ms

cache = {}
total_chunk_num = int((len(speech) - 1) / chunk_stride + 1)
for i in range(total_chunk_num):
    speech_chunk = speech[i * chunk_stride:(i + 1) * chunk_stride]
    is_final = i == total_chunk_num - 1
    res = model.generate(input=speech_chunk, cache=cache, is_final=is_final,
                         chunk_size=chunk_size,
                         encoder_chunk_look_back=encoder_chunk_look_back,
                         decoder_chunk_look_back=decoder_chunk_look_back)
    print(res)
chunk_size=[0,10,5]: display granularity 600ms, lookahead 300ms. Set is_final=True on last chunk to flush remaining text.

Voice Activity Detection (Streaming)

from funasr import AutoModel
import soundfile

chunk_size = 200  # ms
model = AutoModel(model="fsmn-vad")

speech, sample_rate = soundfile.read("audio.wav")
chunk_stride = int(chunk_size * sample_rate / 1000)

cache = {}
total_chunk_num = int((len(speech) - 1) / chunk_stride + 1)
for i in range(total_chunk_num):
    speech_chunk = speech[i * chunk_stride:(i + 1) * chunk_stride]
    is_final = i == total_chunk_num - 1
    res = model.generate(input=speech_chunk, cache=cache,
                         is_final=is_final, chunk_size=chunk_size)
    if len(res[0]["value"]):
        print(res)
Streaming VAD output: [[beg, -1]] = speech start, [[-1, end]] = speech end, [[beg, end]] = complete segment.

Speaker Diarization

Speaker diarization works with Paraformer, Fun-ASR-Nano, and SenseVoice. Add spk_model="cam++" to enable it:

from funasr import AutoModel

model = AutoModel(
    model="paraformer-zh",
    vad_model="fsmn-vad",
    punc_model="ct-punc",
    spk_model="cam++",  # or "iic/speech_eres2netv2_sv_zh-cn_16k-common"
)
res = model.generate(input="meeting.wav", batch_size_s=300)
for sent in res[0]["sentence_info"]:
    print(f"[Speaker {sent['spk']}] [{sent['start']}-{sent['end']}ms] {sent['text']}")

API Reference

AutoModel

ParameterTypeDescription
modelstrModel name or local path
devicestr"cuda:0", "cpu", "mps"
vad_modelstrVAD model for long audio segmentation
punc_modelstrPunctuation restoration model
spk_modelstrSpeaker model for diarization
hubstr"ms" (ModelScope) or "hf" (HuggingFace)
ncpuintCPU threads (default: 4)
batch_sizeintBatch size for inference
disable_updateboolSkip version check
disable_pbarboolDisable progress bar

model.generate()

ParameterTypeDescription
inputstr/array/listAudio file path, URL, numpy array, or list
cachedictCache for streaming (pass {} for first call)
hotwordstrRecognition keywords
batch_size_sintDynamic batch total duration (seconds)
languagestrLanguage hint ("auto", "zh", "en", etc.)
is_finalboolLast chunk flag for streaming
return_spk_resboolReturn speaker diarization results
sentence_timestampboolReturn sentence-level timestamps

Output Format

FieldDescriptionModels
textRecognized textAll ASR models
timestamp[[start_ms, end_ms], ...]Paraformer, SenseVoice, Fun-ASR-Nano
sentence_infoList of {text, start, end, spk, timestamp}When spk_model or sentence_timestamp enabled
value[[start_ms, end_ms], ...]VAD models
spk_embeddingSpeaker embedding tensorCAM++, ERes2NetV2
OOM Issues? Reduce batch_size_s, lower max_single_segment_time in vad_kwargs, or use batch_size_threshold_s to force batch=1 for long segments.
